# Snippet Manager

Расширение для VS Code, которое позволяет создавать, редактировать и запускать команды терминала через удобный интерфейс.

## Возможности

- ✅ Создание и редактирование команд
- ✅ Быстрый запуск в терминале
- ✅ Мультиязычный интерфейс (русский/английский)
- ✅ Сохранение команд между сессиями
- ✅ Удобный интерфейс с поиском и навигацией

## Установка

1. Откройте VS Code
2. Перейдите в Extensions (Ctrl+Shift+X)
3. Найдите "Snippet Manager"
4. Нажмите Install

## Использование

1. Откройте панель Command Runner в Activity Bar
2. Нажмите "+" для добавления новой команды
3. Введите название и команду
4. Кликните по названию для запуска в терминале
